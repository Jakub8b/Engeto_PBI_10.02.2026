# Engeto PowerBI Projekt - Analýza Logistiky

## Přehled Projektu

Tento repozitář obsahuje **PowerBI Dashboard** pro analýzu logistických a supply chain dat. Projekt sleduje objednávky, zákazníky, produkty a dodavatele.

## 📊 Struktura Dashboardu

PowerBI projekt obsahuje čtyři hlavní datové tabulky:

### 1. **Objednávky (Orders)**
- Včasné, včasné a zpožděné dodávky
- Status objednávek
- Analýza dodací spolehlivosti

### 2. **Zákazníci (Customers)**
- Informace o zákanicích a jejich umístění (země, město)
- Metody platby
- Příjmy a hodnota zákazníků

### 3. **Produkty (Products)**
- Kategorie produktů a ID
- Ceny a hmotnosti
- Průměrná cena za kategorii
- Stavy zásob

### 4. **Dodavatelé (Suppliers)**
- Informace o dodavatelích a jejich umístění
- Příjmy od dodavatelů
- Cenové údaje

## 📈 Klíčové Metriky

- Výkonnost dodávek (včasnost)
- Segmentace zákazníků podle regionu a příjmů
- Analýza produktů a zásob
- Zhodnocení dodavatelů

## 🛠️ Technologie

- **Nástroj**: Microsoft Power BI Desktop
- **Formát**: `.pbix` (Power BI Project File)
